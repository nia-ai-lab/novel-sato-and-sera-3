# Style Guide

## Narrative Voice
- 地の文: 各 `話` は必ず `第1節` のサトーサイドから入り、同一事件の裏側を `第2節` のセラサイドで返す
- サトーサイド: 基本はサトー寄りの三人称で、朝の空気、走るリズム、季節の手触り、生活感を丁寧に積む
- サトーサイド: 同じ話のセラの防衛処理が生んだ「ちょっとした違和感」を必ず1つ以上入れる。遠くで誰かがよろける、荷物が壁へ当たる、設備の前で人がしゃがむ、看板が不自然に揺れるなど、サトーからは日常的なノイズにしか見えない形にする
- サトーサイド: サトーはその違和感を事件として認識しない。毎回「春風が強いのだろう」「朝は急ぐ人も多い」「機械の調子でも悪いのだろう」など、お気楽で無害な解釈へ丸めて流す
- セラサイド: 直前のサトー側で起きていた同一事件の正体を短く鋭く描き、読者だけが「あの違和感はこれだった」と分かるように結び直す
- 会話文: 量は多すぎず、関係性より空気を支えるために使う。サトーは穏やか、セラは短く辛辣
- 用語運用: セラの服装は初出のみ `可愛らしいメイド風ジョギングウェア` と書き、その後は `メイド風ジョギングウェア` と短く呼ぶ
- テンポ: 桜、川風、汗が引く感じなどの静かな描写はゆるやかに、護衛処理は短く確実に切る

## Reading Experience
- スマホで読みやすい段落長: 2〜4文程度を基本にする
- 1話の気持ちよい流れ: 第1節でサトーののどかな結論を置き、第2節でその裏にあった危険と処理を明かす
- 次を読みたくなるフック: サトーの何でもない感想と、セラの辛辣な独白の落差
- 節の役割: 第1節は表面、第2節は裏面として働かせる。1話の中で別事件を増やさない
- 読者への情報差: サトーは違和感を日常へ回収し、読者はセラ側でその意味を知る。この情報差を各話で必ず成立させる

## Image Direction
- タイトル画像: サトーの気持ちいい朝ランと、極秘並走するセラの守護性を同居させる
- キャラクター画像: 二作目の既存サトー像とセラ像を元に、この作品専用のジョギング版立ち絵を先に生成する。顔立ち、髪色、体格解釈をぶらさない
- 節画像: 各 `節` を1枚で扱う。各 `話` の `第1節` はサトー主体、`第2節` はセラ主体で固定する
- サトーサイド画像: サトーを主役にし、4月初旬向けジョギングウェアを明確に見せる。セラはいても遠景や遮蔽物の陰に置き、あくまで気配に留める
- セラサイド画像: セラを主役にし、メイド風ジョギングウェアと猫耳カチューシャ型の警護支援ユニットを必ず見せる。敵とのアクションを明確にしつつ、サトーは後景で小さく、危険に気づいていない状態を守る
- 節画像共通: 都内の川沿い、橋、神社、ベーカリー、マンション周りの空気をまず成立させ、同じ話の2枚では背景・光・小道具の連続性を崩さない
- 節画像共通: すべての節画像とタイトル画像は、この作品専用に生成した `sato-jogging` と `sera-jogging` の立ち絵を参照画像として使い、服装差分を起こさない

## Continuity Rules
- 固定設定: サトーは最後までセラの正体に気づかない。セラはサトーの朝の気分を守ることを最優先する
- 固定設定: セラの対処は全編を通して非致死に限る。敵は気絶、睡眠、拘束、行動不能化で無力化し、殺傷しない
- 固定設定: 各話の `第1節` と `第2節` は同じ出来事の表裏であり、出来事の順序は動かさない
- 固定設定: サトーは全節で初春向けジョギングウェアを着る。セラは全節でメイド風ジョギングウェアと猫耳カチューシャ型の警護支援ユニットを着用する
- 固定設定: 川名や橋名や街区名など導線に関わる固有名は必要に応じて架空にする
- 後から変えてよい設定: 都内協力ラインの詳細、敵組織の固有名、サトーのジョギング頻度の細部
