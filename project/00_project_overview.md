# Project Overview

## Logline
45歳バツイチ独身のサトーこと佐藤一は、4月初旬の東京で朝のジョギングに出る。本人は桜吹雪の川沿いを気ままに走っているだけだが、その裏では猫耳カチューシャ型の警護支援ユニットを付けた凄腕護衛セラが、可愛らしいメイド風ジョギングウェアで極秘に並走し、河川敷と橋と街区に散った暗殺者たちを無音で処理し続けている。

## Audience
- 想定読者: 季節感のある日常描写と、気づかれない護衛アクションの落差を楽しみたい読者
- 読後感: 「あの気持ちいい朝ランは、ここまで守られていたのか」という可笑しさと、サトーの慎ましい優雅さが残る
- 連載ペース: 章ごとに確認しながら進める創読形式

## Creative Intent
- この作品でやりたいこと: 4月初旬の東京の朝の冷たさ、桜が舞う都内の川沿いの抜け感、走り終えたあとのささやかな満足を、裏の並走護衛劇と同時進行で描く
- 絶対に入れたい要素: 桜吹雪の川沿い、サトーの4月初旬向けジョギングウェア、セラの可愛らしいメイド風ジョギングウェア、猫耳カチューシャ型の警護支援ユニット、河川敷と橋の風景、走り終えたあとの軽い朝食
- 避けたい要素: スポ根化しすぎる展開、露悪的な暴力、サトーが危機を察知して超人的に動く展開、セラの正体が雑に露見する展開

## Product Intent
- 創読としてどんな体験にしたいか: 読者はサトーと一緒に気持ちよく走りながら、節ごとに「いま裏でどの危険が消されたか」を静かに発見していく
- 画像生成の方針: サトーとセラのキャラクター画像は二作目の既存画像をそのまま継承する。背景、節画像、タイトル画像は今回のジョギング編へ全面更新し、全12節24画像を新規生成する
- モバイル Web での見せ方: 河川敷、橋、土手の坂、遊歩道、朝の部屋など縦画面で奥行きが出る構図を多めにし、春の軽さと秘匿警護の緊張感を同居させる

## Approval Context
- 既存作品から継承するもの: サトーとセラの基本設定、キャラクター画像、現代スパイ日常コメディとしての世界観
- 今回置き換えるもの: タイトル、プロット、章立て、本文、背景イメージ、節画像、タイトル画像、Web公開内容
- 本文開始の前提: ユーザが二作目と同構成での全面差し替えと、一気通貫での画像生成・docsビルドを希望している

## Current Focus
- 今回進める範囲: ジョギング編の設計同期、3章6話12節の本文作成、画像定義更新、背景/節/タイトル画像生成、Webビルド
- 未確定事項: 次回作でサトーの運動習慣をどこまで広げるか、河川敷周辺の協力者を固定キャラ化するか、敵対勢力が都内生活圏へどこまで食い込んでいるか
- 次に決めること: 生成画像の仕上がり確認と、必要なら本文とプロンプトの微調整
