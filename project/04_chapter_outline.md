# Chapter Outline

## Release Strategy
- 1回で見せる分量: ジョギング編として3章6話12節を同規模でまとめ、章ごとに画像とWeb確認ができる状態にする
- 章あたりの体験: 走り出し、中盤の気持ちよさ、走り終わりの余韻で空気を変えつつ、サトーの気ままさとセラの無音護衛を並走させる
- 話あたりの役割: 1話につき1事件だけを扱い、第1節でサトー側の表面、第2節でセラ側の裏面を描く
- 節あたりの役割: 第1節は必ず `サトーサイド`、第2節は必ず `セラサイド` に固定する。2節は同時刻の同一事件であり、別事件を混ぜない
- 話内連動ルール: `第1節` のサトーには、`第2節` のセラの防衛処理が生んだ「ちょっとした違和感」を必ず先に見せる。サトーはそれをお気楽に日常の出来事として解釈し、事件とは気づかない
- 画像生成の最小単位: 節

## Initial Package
- 初回承認前に確定するもの:
  - 全体プロット
  - 主要キャラクター
  - 主要キャラクター画像
  - 背景イメージ
