# Chapter Outline

## Release Strategy
- 1回で見せる分量: ジョギング編として3章6話12節を同規模でまとめ、章ごとに画像とWeb確認ができる状態にする
- 章あたりの体験: 走り出し、中盤の気持ちよさ、走り終わりの余韻で空気を変えつつ、サトーの気ままさとセラの無音並走護衛を並走させる
- 話あたりの役割: 1話ごとに準備、加速、給水、折り返し、帰路、クールダウンなどひとつの行為のまとまりを扱う
- 節あたりの役割: 1節は必ず1事件だけを扱い、その中を `サトーパート` と `セラパート` に分けて同時刻の出来事を重ねる
- 節内連動ルール: `サトーパート` には必ず、同節の `セラパート` で起きる防衛処理の結果が「ちょっとした違和感」として先に見えるようにする。サトーはそれをお気楽に日常の出来事として解釈し、事件とは気づかない
- 画像生成の最小単位: 節内パート

## Initial Package
- 初回承認前に確定するもの:
  - 全体プロット
  - 主要キャラクター
  - 主要キャラクター画像
  - 背景イメージ
