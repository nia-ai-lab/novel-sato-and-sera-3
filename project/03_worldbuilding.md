# Worldbuilding

## Setting Summary
現代日本によく似た世界。東京の都心寄りの住宅街から、架空の都市河川「白鷺川」の河川敷と橋を回る朝のジョギングコースへすぐ出られる距離感のなかで、一般市民の朝ラン風景と国家規模の保全任務が同時進行している。今回の主舞台は、サトーの都内マンション、白鷺川の桜並木、河川テラス、橋梁部、周辺のベーカリーと住宅街。

## Key Background Concepts
- 初期確認で見せる背景 01: 4月初旬の都内河川敷。桜吹雪、薄い朝日、長い遊歩道、遠景の橋
- 初期確認で見せる背景 02: 川風の抜ける橋と河川テラス。給水機、ベンチ、ランナーが点在する朝の都市風景
- 初期確認で見せる背景 03: ジョギングから戻る住宅街と朝の部屋。ベランダ、白い湯気、ベーカリー袋、静かな生活感

## Rules
- 魔法・技術: 超常要素はない。すべて現代的な諜報、監視、暗器、事故偽装、通信妨害で成立する
- 社会: 河川敷や橋では一般のランナーや通勤前の人々が平穏に過ごしているが、国家安全保障の最深部ではサトーを守る秘匿任務が続いている
- 国家機密: 防衛の切り札となる極秘システムは、最終起動に登録済み本人の全身生体認証を必要とする。開発時の事故でサトーの試験用データが本番機に残ったままになり、現在も彼だけが最後の鍵になっている
- 隠蔽構造: 関係官僚と実務担当者は不祥事化と政権批判を恐れ、登録事故を公表せず、非常時のみ秘密裏にサトーを確保すればよいという危うい運用で問題を棚上げしている
- 国際情勢: 第三国はこの秘密を把握しており、防衛システムを無力化するためサトーの殺害、拉致、監禁を狙う
- 地理: 東京の住宅街から徒歩圏に白鷺川の河川敷があり、両岸の遊歩道、橋、テラス、階段、ベーカリー街区が連続している
- 季節: 4月初旬の朝。空気はまだ少し冷たく、日向だけが早く温まる。桜は満開から散りはじめで、風が吹くたび遊歩道に花びらが流れる
- 建築: 住宅街は分譲マンションと低層住宅が混ざる。河川周りは整備されているが観光地化しすぎず、日常の生活インフラとして存在する
- 禁則: 現実の東京らしさは残すが、駅名や川名や橋名など固有の導線は必要に応じて架空名を使い、過度な観光描写にしない
- 叙述方針: この国家機密は設定上の正本として保持し、本文では直接説明しない。警護の過剰さ、敵の執着、サトーの過去を知る者の言い淀みなどで、作品ごとに同じ方向へ匂わせる

## Sensory Notes
- 色: 朝の河川敷は淡い青灰と桜色、橋の上は銀と薄い空色、部屋へ戻った朝は白と木目とコーヒー色
- 質感: 吸汗素材の服、薄いウインドブレーカー、川風、舗装路、花びらの乾いた軽さ、紙袋、紙コップ
- 音: ランニングシューズの接地音、川風、橋の下の反響、自転車のベル、遠い通勤電車、ベーカリーのトング、湯の沸く音
- 匂い: 冷えた朝の川、桜の青い匂い、スポーツドリンク、焼きたてのパン、コーヒー、洗いたての部屋着
