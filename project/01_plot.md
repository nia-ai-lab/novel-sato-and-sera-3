# Plot

## Core Conflict
- 主人公: サトー（本名: 佐藤一 / さとうはじめ）。本文中の表記はサトー。45歳、バツイチ独身、子なし。東京で静かに暮らす中年男性
- 欲望: 4月初旬の朝に都内の川沿いを気持ちよく走り、桜吹雪と川風を味わったあと、満足して軽い朝食を取ること
- 障害: 本人の知らないところで、国家防衛の極秘システムを起動できる最後の鍵として監視され、開けた河川敷と橋梁部を狙って暗殺者に狙われる
- 失敗コスト: サトーが死ぬか敵対勢力に確保されれば国家防衛の切り札が非常時に使えなくなるだけでなく、本人が大事にしている何気ない朝の自由まで暴力で汚される

## Arc Overview
### Beginning
早朝。サトーは都内のマンションで4月初旬向けのジョギングウェアに着替え、薄い冷気の残る街へ出る。白鷺川の河川敷へ向かう本人の少し浮き立った気分のすぐ近くで、セラは猫耳カチューシャ型の警護支援ユニットを付け、可愛らしいメイド風ジョギングウェアで極秘に並走を開始する。

### Middle
朝日が高くなるにつれ、白鷺川沿いの遊歩道と橋はランナーや整備作業員の気配で満ちていく。サトーは桜吹雪の土手、風の抜ける橋、河川テラスの給水所を心地よく巡るが、その背後ではセラがワイヤー、毒針、監視機器、待ち伏せ班を無音で無力化し、朝ランの気分そのものを守り抜く。

### Ending
折り返して街区へ戻ったサトーは、ベーカリーでパンとコーヒーを買い、軽くクールダウンしてから部屋へ帰る。ベランダに立つ朝の満足感の裏で、セラは最後の監視役まで片づけ、サトーに「今日も気持ちよく走れた」という結論だけを残して任務を終える。

## Hidden Truth
- サトーはかつて、国家防衛の切り札となる極秘システムの開発責任者だった
- 開発中、本来は試験専用だったサトーの全身生体認証データがテスト環境に登録され、そのまま事故で本番機へ残留した
- その防衛システムは最終起動時にも登録済み本人の全身生体認証を要求するため、正式運用後もサトーが事実上の最後の鍵になっている
- 政府内部では不祥事の発覚を恐れてこの事実を隠蔽し、「非常時にだけ密かにサトーを呼べばよい」と問題を先送りしてきた
- その秘密は第三国へ漏洩しており、敵対勢力は防衛システムを麻痺させるためサトーの殺害または確保を狙っている
- サトー自身は気ままに日常を過ごしているのだが、自分が国家防衛の鍵になっているとは知らない
